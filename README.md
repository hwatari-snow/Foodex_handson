# Foodex Handson

お惣菜（デリカ）AIエージェント ハンズオン

Snowflake の AI 機能を使って、お惣菜の売上・在庫・気象データを分析できる AI エージェントを構築します。

## 含まれる商品カテゴリ

- **揚げ物**: 唐揚げ、とんかつ、コロッケ、フライ類
- **和惣菜**: 煮物、ひじき、きんぴら、だし巻き卵
- **中華惣菜**: 麻婆豆腐、エビチリ、餃子、春巻き
- **洋食惣菜**: パスタ、グラタン、ハンバーグ、カレー
- **弁当・おにぎり・寿司**: 各種弁当、おにぎり、寿司類
- **季節商品**: 恵方巻き、桜餅、冷やし中華、うなぎ、おでん、おせちなど

## セットアップ

```sql
-- setup_database.sql を実行してください
```

## ファイル構成

- `FOODEX_AI_HANDSON.ipynb` - ハンズオン用ノートブック
- `setup_database.sql` - データベースセットアップスクリプト
- `csv/` - サンプルデータ
  - `product_master.csv` - 商品マスタ（94商品）
  - `pos_data.csv` - POSデータ（2022-2026年）
  - `inventory_data.csv` - 在庫データ
